# PTU\_TEAM Repository Overview

평택대 졸업 작품 프로젝트

## 프로젝트 개요

*   주제

    대학생들이 정보를 공유하고 소통할 수 있는 커뮤니티 사이트



## 주요 기능

* 회원가입 및 로그인
* 게시판 기능 (CRUD)
* 댓글 기능 (CRUD)
* 키워드 검색
* 조회수 및 좋아요 조회
